LocalStorage Guide

LocalStorage is a part of the Web Storage API.

It allows you to store data in the form of key-value pairs in the user's browser.

Data stored in LocalStorage persists even after the browser is closed and reopened.

It has a storage limit of around 5MB per domain (this can vary slightly between browsers).

Basic Operations

  1. Storing Data
    2. localStorage.setItem('key', 'value');

    For example, to store a username:

    localStorage.setItem('username', 'JohnDoe');
  2. Retrieving Data
    3. const value = localStorage.getItem('key');

    For example, to retrieve the stored username:

    const username = localStorage.getItem('username');
  3. Removing Data
    4. localStorage.removeItem('key');

    For example, to remove the stored username:

    localStorage.removeItem('username');
  4. Clearing All Data
    5. localStorage.clear();
